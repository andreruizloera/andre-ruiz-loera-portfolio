# Andre Ruiz Loera — Portfolio

> Personal portfolio site for Andre Ruiz Loera.

**Live:** [andreruizloera.github.io/andre-ruiz-loera-portfolio](https://andreruizloera.github.io/andre-ruiz-loera-portfolio)

---

## Pages

| Route | Description |
|---|---|
| `/` | Home — intro and featured work |
| `/projects` | Side projects and builds |
| `/research` | Research interests and papers |
| `/cv` | CV / résumé |
| `/reading` | Reading notes |
| `/about` | Bio |
| `/contact` | Contact form |

---

## Stack

| Layer | Choice |
|---|---|
| Framework | React 19 + Vite |
| Language | TypeScript |
| UI | shadcn/ui + Tailwind CSS |
| Routing | React Router v6 |
| Hosting | GitHub Pages |

---

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```

Deployed to GitHub Pages via the `gh-pages` branch.
