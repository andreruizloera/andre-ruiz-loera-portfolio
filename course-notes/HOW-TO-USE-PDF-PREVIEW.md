# How to Use PDF Preview in VS Code

## Quick Start

1. **Open the LaTeX file** (e.g., `sisyphus-commentary.tex`)
2. **Press `Cmd+Alt+V`** to open PDF preview

## Layout Options

### Option 1: Side-by-Side (Recommended)
- Open your `.tex` file
- Press `Cmd+Alt+V`
- The PDF appears in a split view
- Drag the PDF tab to the **LEFT** editor group if needed

### Option 2: Full-Screen PDF
1. Click on the PDF preview tab
2. Press `Cmd+B` to hide the sidebar
3. Press `Cmd+J` to hide the panel
4. **Toggle between PDF and LaTeX:**
   - Use `Cmd+1` to focus first editor group (PDF on left)
   - Use `Cmd+2` to focus second editor group (LaTeX on right)

### Option 3: PDF in Separate Window
- Right-click the PDF tab
- Select "Move into New Window"
- Now you can use full-screen on the PDF window

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+Alt+V` | Open/View PDF preview |
| `Cmd+Alt+B` | Build/Compile LaTeX |
| `Cmd+1` | Focus left editor group |
| `Cmd+2` | Focus right editor group |
| `Cmd+B` | Toggle sidebar |
| `Cmd+J` | Toggle bottom panel |
| Double-click in PDF | Jump to source (SyncTeX) |
| `Ctrl+Cmd+F` | Toggle full screen |

## Rearranging Editors

### To Move PDF to Left Side:
1. Click and drag the PDF tab
2. Drag it to the **left edge** of the window
3. Drop when you see the blue highlight on the left

### To Maximize One Side:
- Hover between the editor groups
- Drag the divider to resize
- Drag all the way to one side to maximize the other

## Auto-Compile on Save

Your workspace is configured to auto-compile when you save (`Cmd+S`). The PDF will update automatically!

## Tips

- **Zoom PDF:** Use `Cmd+` and `Cmd-` while focused on PDF
- **Full Screen Mode:** Press `Ctrl+Cmd+F` to go full screen
- **Zen Mode:** Press `Cmd+K Z` for distraction-free writing
- **Exit Zen Mode:** Press `Esc` twice

3. Drag PDF tab to left side
4. Resize so PDF takes ~50-60% of screen
